- "이 상태에서 목표, 실적, 달성률을 1개 행으로 합쳐줘. 가로로 반을 나눠서 좌측 반에 목표와 실적이, 우측 반에 달성률이 나오게 조정해줘."

## 주요 기능

### 1. 초기 화면 (그리드 뷰)
- **핵심지표 섹션**: 6개 카드 (3×2 그리드)
- **기본기 지표 섹션**: 4개 카드 (4×1 그리드)
- 각 카드 표시 정보:
  - 카테고리 번호 및 제목
  - 실적 (current)
  - 목표 (target)
  - 달성률 (achievement)

### 2. 3열 레이아웃 뷰
카테고리 카드 클릭 시 3열 구조로 전환:

#### 1열: 핵심지표 (카테고리 리스트)
- 10개 카테고리 표시
- 선택된 카테고리 하이라이트 (보라색 배경)
- 카테고리 번호 및 제목

#### 2열: 대분류 (서브카테고리 리스트)
- 선택된 카테고리의 서브카테고리 표시
- 각 카드 하단에 **신호등 표시**:
  - 🟢 녹색: 달성률 ≥91% 과제 수
  - 🟡 노란색: 달성률 81~90% 과제 수
  - 🔴 빨간색: 달성률 ≤80% 과제 수

#### 3열: 세부 O/I 과제 (과제 리스트)
- 선택된 서브카테고리의 과제 표시
- 각 과제 카드 구성:
  - **상단**: 과제 번호 및 제목
  - **중단**: 과제 개요 (overview)
  - **하단**: 2단 구조
    - 좌측: 담당 조직명 (배지 형태)
    - 우측: 메트릭 영역
      - 좌측 반: 목표/실적 (세로 배치)
      - 우측 반: 달성률 숫자/진행률 바 (세로 배치)

### 3. 조직 View 기능
3열 레이아웃 화면 상단의 "조직 View" 버튼 클릭 시:

#### 조직 카드 섹션
- 7개 조직 카드를 가로로 배열
- 고정 순서:
  1. MNO AT/DT기획담당
  2. Product&Brand본부
  3. Sales&Marketing본부
  4. Enterprise사업본부
  5. AT/DT센터
  6. Network센터
  7. 안전보건실

#### 조직 카드 구성
- **상단**: 조직명
- **하단**: 2단 구조
  - 좌측: 총 과제 개수 (큰 숫자)
  - 우측: 신호등 표시
    - 🟢 녹색: 달성률 ≥91% 과제 수
    - 🟡 노란색: 달성률 81~90% 과제 수
    - 🔴 빨간색: 달성률 ≤80% 과제 수

#### 필터링 기능
- 조직 카드 클릭 시 해당 조직의 과제만 3열 레이아웃에 표시
- 필터링 적용 범위:
  - 1열: 해당 조직 과제가 있는 카테고리만 표시
  - 2열: 해당 조직 과제가 있는 서브카테고리만 표시
  - 3열: 해당 조직 과제만 표시
  - 신호등도 필터링된 과제 기준으로 재계산
- 동일 조직 카드 재클릭 시 필터 해제
- "전체 View" 버튼으로 조직 View 종료

### 4. 과제 상세 페이지
과제 카드 클릭 시 새 탭에서 열림:

#### 상단 Header
- **Breadcrumb**: 실제 카테고리명 › 서브카테고리명 › 과제명
- **Title Row**: 과제 제목 + 담당 조직 배지

#### Task 섹션
- 과제 개요 (메인 화면과 동일)

#### Target 섹션
- 지표 정의 및 산출 Logic

#### Milestone 섹션
- **월별 Target 및 실적 테이블**:
  - Target: 1~12월 전체
  - 실적: 1~6월만 (7~12월은 "-" 표시)

- **분기별 로드맵 테이블**:
  - Q1~Q4 각 분기별
  - 주요 활동 및 목표

## 주요 함수

### script.js

#### 데이터 관리
- `dashboardData`: 전체 데이터 객체
- `currentState`: 현재 선택 상태 추적
  - `selectedCategory`: 선택된 카테고리 ID
  - `selectedSubcategory`: 선택된 서브카테고리 ID
  - `orgViewActive`: 조직 View 활성화 여부
  - `selectedOrganization`: 선택된 조직명

#### 렌더링 함수
- `renderGridView()`: 초기 그리드 화면 렌더링
- `renderCategoryList()`: 1열 카테고리 리스트 렌더링 (필터링 지원)
- `renderSubcategoryList(categoryId)`: 2열 서브카테고리 리스트 렌더링 (필터링 지원)
- `renderTaskList(subcategoryId)`: 3열 과제 리스트 렌더링 (필터링 지원)
- `renderOrganizationCards()`: 조직 카드 렌더링

#### 뷰 전환
- `showGridView()`: 초기 화면으로 전환
- `showColumnView(categoryId)`: 3열 레이아웃으로 전환

#### 조직 View 관련
- `toggleOrganizationView()`: 조직 View 토글
- `selectOrganization(organization)`: 조직 선택 및 필터링
- `getOrganizationCounts()`: 조직별 과제 수 계산
- `getOrganizationAchievementDistribution(org)`: 조직별 달성률 분포 계산

#### 필터링 함수
- `getFilteredCategories()`: 필터링된 카테고리 반환
- `getFilteredSubcategories(categoryId)`: 필터링된 서브카테고리 반환
- `getFilteredTasks(subcategoryId)`: 필터링된 과제 반환

#### 유틸리티
- `clearTaskList()`: 3열 초기화
- `clearSubcategoryList()`: 2열 초기화
- `openTaskDetail(taskId)`: 상세 페이지 열기
- `setupEventListeners()`: 이벤트 리스너 설정

### detail-script.js

#### 데이터 관리
- `taskDetailsData`: 과제별 상세 데이터 (15개 과제 완료)

#### 렌더링
- `renderTaskDetail(taskId)`: 상세 페이지 전체 렌더링
- `renderMonthlyTable(monthly)`: 월별 테이블 렌더링
- `renderRoadmapTable(roadmap)`: 분기별 로드맵 렌더링

## 인터랙션 및 애니메이션

### 호버 효과
- 카드 hover: `translateY(-5px)` + 그림자 확대
- 버튼 hover: 배경색 반전 + `translateY(-2px)`
- 리스트 카드 hover: `translateX(5px)` + 테두리 색상 변경

### 트랜지션
- 모든 인터랙션: `transition: all 0.3s ease`
- 부드러운 애니메이션 효과

### 선택 상태
- 활성 카드: 보라색 배경 + 흰색 텍스트
- 테두리 하이라이트: border-color 변경

## 사용 방법

### 로컬 실행
1. 파일들을 동일 디렉토리에 배치
2. `index.html`을 브라우저로 열기
3. 프레임워크나 빌드 도구 불필요

### 네비게이션
1. **초기 화면**: 카테고리 카드 클릭 → 3열 레이아웃
2. **3열 화면**:
   - 타이틀 클릭 → 초기 화면 복귀
   - "조직 View" 버튼 → 조직 필터링 모드
3. **조직 View**:
   - 조직 카드 클릭 → 해당 조직 과제만 표시
   - "전체 View" 버튼 → 조직 View 종료
4. **과제 카드**: 클릭 → 새 탭에서 상세 페이지
